## Programming Language :- 

=> Programming language is a formal language that devlopers use to communicate with computers.

=> there are 2 types of Language:

1. Complied Language :
=> it runs the whole at a time.
=> it takes less time to run.
=> it is hard to debug.
EXAMPLE: Java etc .

2. interpreted Language :
=> it runs the code line by line .
=> it takes more time to run .
=> it is easy to debug.
EXAMPLE: Python etc.


* DEBUG : it's the process of finding erros.


## APPLICation :

=> APPLICation is a type of software , that is used to devloped to perform any specific task .

=> it can run on any mobile devices , computers, and even any web browsers.

## browsers : Micrsoft Edge, Safari, Google CHrome, Firefox , Opera mini

## APPLICation TYpes :
1. Web APPLICation
2. Mobile APPLICation
3. Desktop APPLICation

1. Web APPLICation: -
=> It runs inside web browsers.
=> it requires any internet connection.
=> it is platform independent(Operating system).
=> No need to install.

EXAMPLE : google translator, Youtube, etc.

2. Mobile APPLICation:
=> it is devloped to run on smartphones, tabletst.

=> there are 3 types of mobile APPLICation

(a) Mobile web APPLICation :
=> there are so many apps, that run in mobile inside any web browser.
=> access via url
=> it is platform independent.
=> it requires any internet connection.

EXAMPLE: google doc, google sheet. 

(b) Native APPLICation:-
=> it is platform dependent. 
=> it is devloped for a particular platform(Operating system).
=> installed from app store and play store.
=> Full access to device features ( Camera, GPS, Notification)
EXAMPLE: App store, Play store .

(c) Hybrid Application :-
=> it's combination of Mobile web Application and Native application.
=> it is platform independent.
=> it can run as a app or as a website.

EXAMPLE: Youtube, Instagram, Facebook, Whatsapp.
